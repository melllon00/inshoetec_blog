# inshoetec_blog

### 신발 건조기 상품 소개를 위한 랜딩 페이지 제작
- 서버를 사용하지 않은 html, css, js 로만 구성된 정적 사이트
- 이미지 파일을 많이 사용하여 구성
- 개발 초기 제작한 사이트
